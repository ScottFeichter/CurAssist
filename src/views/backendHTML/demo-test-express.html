If you are seeing this it means Express is functioning correctly and CSRF is acquired.
From here you can return to the root, view the csrf token, or test other parts of the backend.

To View The CSRF Token In Cookies

  1. Open DevToos using:
    • Windows: Ctrl + Shift + I
    • Mac: Cmd + Option + I
  2. Then:
    • Click the "Application" tab
    • Expand the "Storage" section
    • Click on "Cookies"

Return To Root

To return to the root click here: http://localhost:5555/
